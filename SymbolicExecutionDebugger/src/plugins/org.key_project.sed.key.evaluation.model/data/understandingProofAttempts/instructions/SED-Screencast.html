Best Practices

  1. Check if all execution paths end with a termination, exceptional termination or loop body termination node. Otherwise symbolic execution has not finished.
  2. Look for unverified nodes indicated by a red cross. For each of them:
    1. Comprehend the truth values to identify what could not be shown.
    2. Inspect the current state to identify bugs. If there is none, the proof can be closed interactively outside of SED.
